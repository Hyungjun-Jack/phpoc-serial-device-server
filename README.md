# PHPoC 시리얼 디바이스 서버

PES-2406 RS232 스마트 확장보드와 PES-2407 RS422/RS485 스마트 확장보드를 사용해서 만든 시리얼 디바이스 서버입니다.

PHPoC블랙 또는 블루의 시리얼 포트를 최대 14개까지 확장할 수 있습니다.
